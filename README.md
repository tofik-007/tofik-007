## 🧪 What I'm Building

**Splunk SOC Home Lab — Ransomware Detection**
Building MITRE ATT&CK–mapped SPL detection queries covering techniques like `T1059.001`, `T1053.005`, `T1547.001`, `T1021.002`, and `T1486`. Configured with a dedicated `soc_lab` index, Sysmon (SwiftOnSecurity config), and PowerShell logging for interview-ready portfolio work.

**DNS & HTTP Log Analysis Guides**
Structured Splunk triage workflows for analyzing DNS and HTTP logs — baseline → narrow → attribute → validate — built around free sample datasets for hands-on SOC practice.

**Wazuh SIEM Lab**
Home lab simulating brute-force detection scenarios (SSH) across Ubuntu and Windows VMs on VirtualBox.
